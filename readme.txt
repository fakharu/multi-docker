How to creat a git repo:
1) On local console:
    git init
    git add .
    git commit -m "comment" 
2) Create repository on GitHub
    On git hub create a repo: https://github.com/
    git remote add origin https://github.com/fakharu/multi-docker.git
    git remote -v
    git push origin master
